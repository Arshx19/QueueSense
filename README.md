# QueueSense 🧠
Smart Queue Management System with ML-powered wait time prediction.

## Tech Stack
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- ML: Python + scikit-learn + Flask
